### Recommended Modifications to Improve Performance

#### 1. Lower Entry Threshold (Currently 70 ‚Üí Suggested 60)

**Current Rule:** IBD score >= 70 for entry
**Problem:** Too restrictive, only 5-7 trades in 2.9 years
**Suggested:** IBD score >= 60 for entry

**Expected Impact:**
- More trades (10-15 vs 5-7)
- Higher market participation (40% vs 20% of days)
- Better capture of sustained trends
- Estimated improvement: +10-15% total return

#### 2. Relax Exit Criteria

**Current Rule:** Exit if IBD < 50
**Problem:** Exits on temporary weakness
**Suggested:** Exit if IBD < 40 AND below 50-day MA

**Expected Impact:**
- Reduce whipsaws
- Hold winners longer
- Estimated improvement: +5-10% total return

#### 3. Add "Trend Override" for Strong Moves

**New Rule:** If in position AND up >20%, only exit on:
- Stop loss (10%)
- Take profit (30%)
- Below 50-day MA for 3 consecutive days

**Expected Impact:**
- Capture parabolic moves
- Current Apple trade (+10.59%) could run to +20-30%
- Estimated improvement: +8-12% total return

#### 4. Use Scaled Position Sizing

**Current:** 100% of capital per trade (aggressive)

- IBD 70-79: 50% of capital
