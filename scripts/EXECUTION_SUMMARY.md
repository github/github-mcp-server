## ⚠️ 关键风险与提示

### 1. 回测的局限性

✓ **优点**
- 历史数据验证了策略可行性
- 证明了高分红品种的稳定性
- 确认了交易周期的有效性

⚠️ **风险**
- 历史表现不等于未来结果
- 市场结构可能发生变化
- 派息政策可能调整
- 流动性可能出现变化

### 2. 成本与税费

**未考虑的因素**
- 交易手续费 (通常0.03%-0.1%)
- 印花税 (中国A股: 0.1%)
- 个人所得税 (分红税率: 20%-35%)
- 汇兑费用 (外汇交易费)

**影响估算**
```
假设成本: 2% (手续费+税费)
实际月度收益 ≈ 理论收益 × (1 - 0.02)

中国: 9.68% × 0.98 ≈ 9.49%
美国: 15.26% × 0.98 ≈ 14.95%
```

### 3. 流动性风险

**风险等级**
- 🟢 低风险: JEPI, XYLD, SDIV (美国主流ETF)
- 🟡 中等风险: 中国A股大盘股 (601988, 601398等)
- 🔴 高风险: H股和中小市值品种

### 4. 派息连续性

**中国策略风险**

- 行业政策可能影响派息能力

## 📈 绩效监控指标

### 每笔交易

```
计划收益% = 分红率% + 预期升值%
实际收益% = (卖价 - 买价 + 分红) / 买价 × 100%
差异 = 实际 - 计划

追踪: 如果差异 > 1%，分析原因
```

### 月度汇总

```
月度实际收益 = Σ 单笔实际收益
月度收益率 = 月度实际收益 / 本月初资本 × 100%
月度对标 = 月度收益率 / 预期月收益 × 100%

目标: 对标率 ≥ 80%
```

### 年度评估

```
年度累计收益 = Σ 月度实际收益
年化收益率 = (期末资本 - 期初资本) / 期初资本 × 100%
夏普比率 = 年化收益 / 波动率

评估: 与预期183.06% (美国) / 116.14% (中国) 对比
```
