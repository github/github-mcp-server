# 中国股息轮动策略执行指南 (China Dividend Rotation Strategy - Complete Guide)

## 概览 (Overview)

**策略名称:** 中国股息轮动策略 v1  
**目标:** 最大化RMB投资组合收益通过高频率股息捕获  
**时间框架:** 3-5天持仓周期  
**预期收益:** 0.3-0.8% 每个周期 (月化 2-5%)  
**风险等级:** 中等 (Market risk low, dividend cut risk medium)

---

## 策略核心原理 (Core Strategy)

### 基本逻辑:
```
买入日期 = 除权日 - 2个交易日
持仓日期 = 买入日期 至 卖出日期 (3-5天)
卖出日期 = 除权日 + 1个交易日
```

### 为什么有效:
1. **除权套利**: 获取分红同时享受除权前的最后机会
2. **快速轮动**: 3-5天快速周期，降低市场风险
3. **股息稳定**: 中国蓝筹股分红相对稳定
4. **多市场机会**: A股 + H股 + ETF = 更多交易机会

## 执行步骤 (Execution Steps)

### 第一步: 账户准备 (Account Setup)

**A股账户:**
```
1. 开设证券账户 (如果没有):
   - 携带身份证到证券公司营业部
   - 或通过手机App (同花顺, 东方财富等)
   
2. 转入资金 ¥50,000+ (推荐)
   - 银行卡关联到证券账户
   - 支持快速转账
   
3. 设置交易参数:
   - 佣金: 协商至 ≤0.002% (通常0.1%)
   - 融资杠杆: 不推荐使用
   - 风控限额: 设置单笔交易限额
```

**H股账户 (港股通):**
```
1. 开通港股通权限:
   - 需在A股证券账户基础上申请
   - 资金要求: ¥500,000+ (通常)
   - 部分券商降低门槛至¥50,000
   
2. 配置汇兑参数:
   - 监控 HKD/CNY 汇率
   - 设置合理的卖出目标 (考虑汇率)
```

### 第二步: 日常监控 (Daily Monitoring)

**每天09:00:**
```bash
# 执行脚本获取最新分红日历
python dividend_rotation_china_v1.py \
  --lookahead 60 \
  --output China_Daily_Plan.md
```

**检查清单:**
- ☐ 记录今日应买入的股票 (T+2前)
- ☐ 确认分红除权日未变更
- ☐ 监控股价是否有突然变动 (>3%)
- ☐ 确认账户资金充足

**每天14:30 (收盘前30分钟):**
- ☐ 确认所有建仓完成
- ☐ 设置卖出计划 (第二天执行)
- ☐ 检查所有头寸是否安全

### 第三步: 建仓 (Entry)

**买入策略:**

```
买入时机: 除权日T-2天
买入时间: 09:30 开盘 (最好的流动性)
买入数量: 
  - 单个头寸: 不超过 ¥10,000
  - 每日总投入: 不超过可用资金的33%
  - 最多同时3-4个头寸

买入价格:
  - 使用市价单 (实时成交，快速)
  - 或限价单 (当日有效)
  - 避免盘中波动，坚守开盘买入
```

**A股买入示例:**
```
股票: 601988 (中国银行)
除权日: 2025-12-05
买入日: 2025-12-03 (T-2)
买入价格: ¥4.50 (实时价格)
数量: 2,000股
总投入: ¥9,000
```

**H股买入示例:**
```
股票: 01288.HK (农业银行H股)
除权日: 2025-12-13
买入日: 2025-12-10 (T-2)
买入价格: HKD 3.20 (实时价格)

```

### 第四步: 持仓管理 (Position Management)

**持仓期间 (3-5天):**

每日09:00检查:
```
□ 当前价格 vs 买入价格 (目标: 持平或+0.2%)
□ 账户权益变化
□ 有无突发利空消息
□ 分红公告是否有变化
```

**风险监控:**
- 如价格 -2%: 立即止损卖出
- 如分红被取消/大幅削减: 立即卖出
- 如突发重大利空: 按规则止损

### 第五步: 平仓 (Exit)

**卖出策略:**

```
卖出时机: 除权日T+1天
卖出时间: 09:30-10:00 (最好流动性)
卖出价格: 
  - 市价单成交 (优先)
  - 或设置目标 +0.3% 至 +1.0%
  - 不贪心，快速成交最重要

确认卖出:
  - 检查成交单据
  - 记录卖出价和时间
  - 计算P&L
```

**A股卖出示例:**
```
股票: 601988 (中国银行)
卖出日: 2025-12-05 (T+1, 除权后)
卖出价格: ¥4.53 (目标 +0.7%)
数量: 2,000股
总卖出: ¥9,060
分红收入: 2,000 × ¥0.033 = ¥66
总收益: ¥9,060 + ¥66 - ¥9,000 = ¥126 (1.4% 在3天内)
```

## 成本与税收 (Costs & Taxes)

### 交易成本 (Trading Costs)

**A股:**
```
单向手续费: 0.05% - 0.1% (通常)
卖出印花税: 0.1% (政府税，固定)

单笔 ¥10,000 成本估算:
- 买入: ¥10 - ¥20 (0.1-0.2%)
- 卖出: ¥20 + ¥10 = ¥30 (含0.1%印花税)
- 总成本: ¥40 - ¥50 (0.4-0.5% 往返)
```

**H股:**
```
佣金: 0.08% - 0.15% (通常)
汇兑成本: 0.2% - 0.5% (券商报价差)

单笔 ¥10,000 (等值HKD)成本估算:
- 买入: ¥16 - ¥20
- 卖出: ¥24 - ¥40
- 汇兑: ¥20 - ¥50
- 总成本: ¥60 - ¥110 (0.6-1.1%)
```

### 分红税 (Dividend Taxes)

**中国个人投资者:**
```
A股股息税: 
  - 持股 ≥12个月: 免税
  - 持股 1-12个月: 10%
  - 持股 <1个月: 20%
  
对本策略的影响: 
  - 3-5天持仓 = 20%分红税
  - 分红 ¥66 → 实得 ¥52.8 (税后)
  
建议: 组合中保留一些长期持仓 (≥1年) 
     来平衡短期交易的税负
```

**H股分红税:**
```
一般不征税 (但有预扣税)
具体情况取决于你的纳税地位
建议咨询您的税务顾问
```

## 关键风险与对策 (Risks & Mitigation)

### 风险1: 分红取消或削减 (Dividend Cut)

| 风险等级 | 原因 | 信号 | 对策 |
|---------|------|------|------|
| 中等 | 公司经营不佳 | 盈利下滑 -20% | 立即止损卖出 |
| 中等 | 政策变化 | 银行监管收紧 | 快速轻仓 |
| 低 | 一次性调整 | 公告但分红保留 | 观望，保持头寸 |

**监控:**
```
每周查看公司公告:
- 季度财报 (分红预期)
- 监管政策变化
- 行业重大动向

订阅: 巨潮资讯、同花顺资讯
```

### 风险2: 价格波动 (Price Gap)

**可能的情况:**
```
情况1: 分红突然取消 → 价格跳空下跌
对策: 设置 -2% 止损

情况2: 市场调整 → 短期波动 ±1%
对策: 忽略，坚守计划到卖出日

情况3: 利空消息 → 突然下跌 >2%
对策: 是否是持久影响? 
      - 是 → 止损
      - 否 → 继续持有
```

### 风险3: 流动性不足 (Low Liquidity)

**特别是H股:**
```
问题: 某些小盘H股买卖价差大
影响: 实际成交价可能较差

预防:
- 只交易日均成交量 > ¥1000万的品种
- 避免盘尾建仓 (卖方容易被动止损)
- 优先选择主流蓝筹H股
```

### 风险4: 汇率风险 (FX Risk)

**H股特有:**
```
问题: HKD/CNY 汇率波动
例: 如果HKD升值 2%, 可能抵消分红收益

对策:
1. 卖出时同时结汇 (不要持有HKD)
2. 选择低分红但流动性好的H股
3. 设置汇率止损 (HKD升值 > 1%)
```

## 案例分析 (Case Studies)

### 案例1: 成功的A股轮动

```
股票: 工商银行 (601398)
账户规模: ¥100,000

Day 2 (2025-12-04, T-1):
- 价格: ¥4.73 (略跌)
- 头寸价值: ¥10,406
- 浮亏: -¥54 (-0.5%)
- 操作: 无操作，继续持有

Day 3 (2025-12-05, 除权日):
- 早盘除权: ¥4.72 (除权)
- 获得分红: 2,200 × ¥0.028 = ¥61.6
- 头寸价值: ¥10,384 + ¥61.6 = ¥10,445.6

Day 4 (2025-12-06, T+1):
- 卖出: 2,200股 @ ¥4.78 = ¥10,516
- 手续费和印花税: ¥20 (0.2%)
- 净收入: ¥10,496

总收益: ¥10,496 - ¥10,460 + 分红 = ¥36
收益率: 0.34% 在3天内 (年化: ~41%)
```
