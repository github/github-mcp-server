## 📋 账户准备清单 (Account Setup Checklist)

### A股账户 (A-Share Account)
```
□ 开设证券账户
  └─ 推荐券商: 同花顺、华泰、中信、招商证券
  
□ 完成身份验证
  └─ 需要: 身份证、银行卡、手机号
  
□ 绑定银行卡
  └─ 用于充值/提现
  
□ 转入启动资金
  └─ 推荐: ¥50,000+ 
  └─ 最少: ¥10,000 (小规模测试)
  
□ 下载交易App
  └─ 推荐: 同花顺、东方财富、券商官方App
  
□ 设置交易参数
  └─ 佣金率: 协商至 ≤0.002% (通常0.05-0.1%)
  └─ 日交易上限: 无特殊要求
```

### H股账户 (H-Share/港股通)
```
□ 在A股账户基础上申请港股通
  └─ 需要: 5万元资金证明 (某些券商降至1万)
  └─ 时间: 3-5个工作日
  
□ 绑定HKD兑换
  └─ 大多数券商自动支持
  └─ 汇率: 实时市场价
```

## 🎯 第一笔交易准备 (First Trade Setup)

### 步骤1: 确认你的第一笔交易

查看生成的 `My_China_Plan.md`, 找到：
```
✓ 最近的买入日期 (下一个 T-2 天)
✓ 要买的股票代码和名称
✓ 推荐买入金额 (¥5,000 - ¥10,000)
✓ 除权日期 (记住这个日期!)
```

### 步骤2: 验证账户资金

```
打开你的App:
□ 进入"账户" → 查看可用资金
□ 确保金额 ≥ 买入金额 + 手续费
□ 确保账户正常 (无风控限制)
```

### 步骤3: 下单买入

```
买入指令:
1. 搜索股票代码 (例如: 601988)
2. 点击"买入"
3. 输入数量和价格:
   └─ 价格: 使用"市价" (最快成交)
   └─ 数量: 自动计算 (总额 ¥10,000)
4. 确认 → 提交
5. 等待成交 (通常 <1秒)
6. 保存成交单据截图
```

### 步骤4: 设置卖出提醒

```
在你的App中:
□ 设置日期提醒 (卖出日期)
□ 或在日历中标记
□ 关键: 不要忘记卖出日期!
```

## 💰 预期收益计算 (Return Calculation)

### 保守方案 (Conservative)

```
初始资本: ¥50,000
月轮动: 4次
单次收益: 0.3%
月收益: 4 × 0.3% = 1.2% = ¥600
年收益: 1.2% × 12 = 14.4% = ¥7,200
```

### 稳健方案 (Moderate)

```
初始资本: ¥50,000
月轮动: 8次
单次收益: 0.5%
月收益: 8 × 0.5% = 4% = ¥2,000
年收益: 4% × 12 = 48% = ¥24,000
```

### 积极方案 (Aggressive)

```
初始资本: ¥50,000
月轮动: 12次 (日轮动)
单次收益: 0.8%
月收益: 12 × 0.8% = 9.6% = ¥4,800
年收益: 9.6% × 12 = 115% = ¥57,500
```

⚠️ 注意: 积极方案需要更高的风险管理技能

## 📞 常见问题 (FAQ)

**Q: 我没有证券账户怎么办?**
A: 下载任何大券商的App(同花顺、东方财富等)，按提示开户。10分钟完成。

**Q: 最少投入多少钱?**
A: ¥5,000就可以开始。推荐¥50,000以上获得更多机会。

**Q: 分红什么时候到账?**
A: T+1天自动转入现金账户。可以继续买入其他股票。

**Q: 可以融资杠杆吗?**
A: 不推荐。这个策略只需要简单持仓，融资会增加风险。

**Q: 亏了怎么办?**
A: 设置-2%止损，快速切换到下一个机会。不要死守。

**Q: 一个月要做多少笔交易?**
A: 根据你的风险偏好。保守4次，稳健8次，积极12次。
