# Setup Guide for the GitHub MCP Server

## Prerequisites
- Make sure you have Git installed.
- Have a GitHub account to access the repository.

## Setting Up Visual Studio Code (VS Code)

1. **Download and Install VS Code:**
   - Visit [Visual Studio Code Download Page](https://code.visualstudio.com/Download).
   - Download the installer for your operating system and follow the installation instructions.

2. **Install Extensions:**
   - Open VS Code.
   - Go to the Extensions view by clicking on the Extensions icon in the Activity Bar.
   - Install the following recommended extensions:
     - C/C++ (by Microsoft)
     - Python (by Microsoft)
     - Debugger for Chrome (if applicable)

3. **Clone the Repository:**
   - Open a terminal in VS Code (`` Ctrl + ` ``).
   - Run the command:
     
     ```
     git clone https://github.com/scutuatua-crypto/github-mcp-server.git
     ```
   - Navigate into the cloned directory:
     
     ```
     cd github-mcp-server
     ```

4. **Open the Project:**
   - Open the cloned project in VS Code by selecting 'File' > 'Open Folder...' and navigating to the `github-mcp-server` directory.

5. **Configure Project Settings:**
   - Ensure that your project settings are configured properly for your environment. You may create or modify configuration files as needed.

## Setting Up Other IDEs (if applicable)

1. **Download and Install Other IDEs (Optional):**
   - Depending on your preference, you can choose to set up IntelliJ IDEA, Eclipse, or any other IDE you prefer.

2. **Open the Project:**
   - For IntelliJ IDEA, select `Open` and choose the `github-mcp-server` directory.
   - For Eclipse, you may need to import the project as a Gradle or Maven project based on your setup.
